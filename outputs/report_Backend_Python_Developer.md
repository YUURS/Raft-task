## План обучения

### 1 Foundation (30 дней)

**Темы**: Python Fundamentals, SQL Basics, Django Fundamentals

**Ресурсы**:
- Python Tutorial (Official) (документация)
- SQLZoo (онлайн курс)
- Django Tutorial (официальная документация)

**Milestone**: Понимание основных концепций Python, SQL и базового Django. Умение писать простые запросы к БД и создавать базовые веб-приложения.

### 2 Practice (45 дней)

**Темы**: Django Models & Views, Django REST Framework Basics, API Design

**Ресурсы**:
- Django REST Framework Tutorial (онлайн курс)
- Django Documentation - Models (документация)
- REST API Design Principles (статья/книга)

**Milestone**: Умение разрабатывать REST API с использованием Django REST Framework, модели данных и создавать CRUD операции.

### 3 Portfolio (30 дней)

**Темы**: Advanced Django Features, Testing, Deployment

**Ресурсы**:
- Django Documentation - Advanced Features (документация)
- TestDriven.io (онлайн курс)
- Heroku Deployment Guide (документация)

**Milestone**: Разработка полноценного веб-приложения с использованием Django, включая тестирование и развертывание на облачной платформе.

## Карьерный совет

### Быстрые решения (2-4 недели)

- **Docker**: Изучить основы Docker и научиться создавать Dockerfile для простого Django приложения. (2-4 недели)
- **Git**: Освоить базовые команды Git для управления версиями кода. (1 неделя)
- **FastAPI**: Сравнить FastAPI с Django и создать простой проект для сравнения производительности (2-4 недели)

### Долгосрочное развитие (3+ месяцев)

- **Kubernetes**: Изучить принципы контейнеризации и развертывание приложений в Kubernetes. (3+ месяцев)
- **AWS/Azure/GCP**: Освоить один из облачных провайдеров и использовать его для развертывания проектов. (6+ месяцев)

## Портфельный проект

**Название**: Simple Task Manager API

**Описание**: REST API для управления задачами (CRUD операции). Приложение позволяет создавать, читать, обновлять и удалять задачи. Используется Django REST Framework и PostgreSQL.

**Технологии**: Python, Django REST Framework, PostgreSQL, Django, Git
