## План обучения

### 1 Foundation (30 дней)

**Темы**: Основы Python и ML, Работа с данными (Pandas, NumPy), Статистика и вероятности

**Ресурсы**:
- Python для Data Science от Google (курс)
- Pandas Documentation (документация)
- Introduction to Statistical Learning (книга)

**Milestone**: Понимание базовых концепций машинного обучения и умение работать с данными на Python.

### 2 Practice (45 дней)

**Темы**: TensorFlow и PyTorch, Разработка и обучение моделей, Оценка и выбор моделей

**Ресурсы**:
- TensorFlow Tutorial (документация)
- PyTorch Tutorials (документация)
- Scikit-learn Documentation (документация)

**Milestone**: Умение строить, обучать и оценивать простые модели машинного обучения с использованием TensorFlow/PyTorch и Scikit-learn.

### 3 Portfolio (30 дней)

**Темы**: Развертывание моделей (MLflow, Docker), Работа с облачными платформами (AWS/Azure/GCP), Оптимизация и масштабирование моделей

**Ресурсы**:
- MLflow Documentation (документация)
- Docker Documentation (документация)
- AWS/Azure/GCP Tutorials (документация)

**Milestone**: Умение развертывать обученные модели на сервере и использовать MLflow для отслеживания экспериментов.

## Портфельный проект

**Название**: Классификация изображений кошек и собак

**Описание**: Разработка модели машинного обучения, которая классифицирует изображения кошек и собак на основе данных, собранных из открытых источников (например, Kaggle).  Использование TensorFlow/Keras для создания и обучения модели, а также развертывание модели на облачной платформе.

**Технологии**: Python, TensorFlow, Keras, AWS/GCP
